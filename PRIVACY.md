# Privacy Policy for NotebookLM Quick Capture

Effective date: April 21, 2026

NotebookLM Quick Capture is a Chrome extension that helps users save a web page URL, page text, highlighted text, or manually pasted text into a NotebookLM notebook selected by the user.

This Privacy Policy explains what information the extension accesses, how it is used, and what is and is not collected by the developer.

## Summary

- The extension does not run a developer-operated backend service.
- The developer does not sell user data.
- The developer does not use user data for advertising.
- The extension only accesses page information and content when the user initiates an action such as `URL first`, `Text first`, `Selection first`, or a side-panel action.
- When a user chooses to save content to NotebookLM, the relevant data is sent directly from the user's browser to Google services used by NotebookLM.

## Information the Extension Accesses

Depending on the user's action, the extension may access the following categories of information:

### 1. Web history and page metadata

The extension may access:

- the current tab URL
- the page title
- related tab state needed to identify the active page or open the target NotebookLM page

This is used only to complete a user-requested capture and to help avoid duplicate saves.

### 2. Website content

The extension may access:

- page text extracted from the current web page
- text highlighted by the user
- text manually pasted by the user into the extension UI

This content is used only to create the NotebookLM source the user requested.

### 3. NotebookLM-related account and notebook data

When the user is signed in to NotebookLM, the extension may retrieve limited NotebookLM data needed to operate, such as:

- available accounts
- available notebooks
- existing source metadata used for duplicate checks

This is used only to let the user choose a destination notebook and to support the save flow.

### 4. Local extension settings and recent history

The extension stores a small amount of data locally in the browser, such as:

- the selected default notebook
- interface settings
- a short recent history of capture actions and results

This local storage is used only to restore preferences and improve repeated user-initiated captures.

## How Information Is Used

The extension uses accessed information only for the following purposes:

- saving a page URL to a selected NotebookLM notebook
- saving copied page text to a selected NotebookLM notebook
- saving highlighted text selected by the user
- showing available notebooks and accounts in the extension UI
- checking existing notebook sources to reduce duplicate captures
- restoring local preferences and recent capture history
- opening or focusing NotebookLM tabs needed to complete the requested action

The extension is not designed or used for:

- advertising
- data brokerage
- profiling unrelated to the extension's single purpose
- creditworthiness, lending, or insurance decisions

## Where Data Goes

### Developer

The developer of NotebookLM Quick Capture does not receive, collect, or store the captured page content on a developer-controlled server.

### Google / NotebookLM

When the user requests a save to NotebookLM, the relevant information is transmitted from the user's browser to Google services used by NotebookLM in order to complete that request.

### Third parties

Other than the services needed to complete the user-requested NotebookLM flow, the extension does not sell or transfer user data to third parties for unrelated purposes.

## Data Retention

- Local settings and recent history remain in the user's browser until the user clears the extension's stored data, removes the extension, or overwrites the stored values.
- Content sent to NotebookLM is retained according to the user's Google / NotebookLM account settings and Google's policies.

## Security

The extension is packaged with its code and does not use remote executable code. It uses Chrome extension permissions and local storage to perform its features.

No method of transmission or storage is perfectly secure, but the extension is designed to minimize data handling and limit access to user-initiated actions.

## User Choices

Users can:

- choose whether to use the extension at all
- choose which notebook receives content
- choose whether to save a URL, page text, highlighted text, or manually pasted text
- remove the extension at any time
- clear browser extension storage using Chrome's extension management tools

## Changes to This Policy

This Privacy Policy may be updated from time to time. If material changes are made, the updated policy will be reflected in this document with a new effective date.
